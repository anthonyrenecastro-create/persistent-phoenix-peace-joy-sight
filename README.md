Symmetry-Modulated DRSSTC

This repository documents a resonant high-voltage hardware project based on a conventional DRSSTC topology with an experimentally testable variation in primary-coil geometry. The main design change is a 12-fold symmetry-modulated primary intended to improve field distribution and reduce localized stress relative to simpler spiral layouts.

The repository contains:

simulation tools for coupling and envelope behavior
printable winding-template outputs
build-oriented design documentation
a validation plan for bench testing

This work should be read as an engineering prototype and modeling effort. Simulation results reported here are preliminary and require experimental confirmation.
